# DevOps o que é?
É uma metodologia o qual unifica as equipes de Desenvolvimento de Software (Dev) e Operações de TI (Ops), servindo como uma ponte entre os processos e ferramentas, eliminando barreiras na entrega de softwares.

# SRE o que é?

SRE (Site Reliability Engineering) é uma prática dedicada a engenharia que aplica princípios de Engenharia de Software para automatizar operações de TI, visando a confiabilidade e escalabilidade de sistemas.
